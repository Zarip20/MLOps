# MLOps MVP

## Задача и описание

Задача: Бинарная классификация клиентов страховой компании по уровню риска (высокорисковые / низкорисковые) на основе их социально-демографических и имущественных характеристик. Используется учебный набор данных Ethiopian Insurance Corporation.

Реализованные обязательные этапы
- Сбор данных: разбиение на батчи, файловое хранилище, расчёт метапараметров.
- Анализ данных: оценка качества, генерация ассоциативных правил (Apriori, 5 правил), очистка.
- Подготовка данных: обработка пропусков, категориальных и числовых признаков.
- Обучение: дерево решений и нейросеть, дообучение MLP.
- Валидация: hold-out, хранение версий моделей и метрик.
- Обслуживание: сериализация лучшей модели.
- Управление: консольные команды, конфигурационный файл, логирование.

Метрики
- Качество данных: полнота, валидность, своевременность, нарушения статических и динамических правил.
- Модели: precision, recall, f1, roc_auc. Лучшая модель выбирается по f1.

Требования

- Python 3.12
Установка зависимостей:
- pip install -r requirements.txt

## Сценарии использования

# Инициализация
python run.py -mode init

- Разбивает исходный CSV (data/motor_data14-2018.csv) на батчи по месяцам.
- Сохраняет список батчей в state.json.

# Обработка каждого батча
python run.py -mode update

- Оценка качества данных, дообучение MLP (partial_fit) и переобучениче дерева на всех накопленных данных.
- Оценка модели по метрикам (precision, recall, f1, roc_auc).

# Инференс на тестовом файле
python run.py -mode inference -file путь_к_файлу

# Формирование отчёта
Создаётся в папке reports
python run.py -mode summary
